# 📝 文本标注系统

一个基于 Flask 的智能文本标注系统，支持中文分词、词性标注和 5 种命名实体识别。

## ✨ 功能特性

### 核心功能
- **文本管理**：支持文件上传（.txt）和手动输入，任务列表管理
- **智能标注**：自动分词、词性标注、命名实体识别、文本分类、情感分析
- **手动标注**：双模式标注（分词模式/实体模式），支持词性修改、实体标注、词语合并
- **知识库管理**：自动学习实体、手动添加、搜索、导出功能
- **数据统计**：可视化图表展示标注进度和实体分布

### 支持的命名实体类型（5种）
- 👤 **人名**：人物名称，如"张三"、"李四"
- 📍 **地名**：地点名称，如"北京"、"上海"
- 🏢 **组织机构**：组织机构名称，如"清华大学"、"阿里巴巴"
- 📅 **时间日期**：时间信息，如"2024年"、"今天"
- 💲 **数值金额**：数值或金额，如"100元"、"1000万"

### 支持的词性标注（12种）
- 实词：名词(n)、动词(v)、形容词(a)、数词(m)、量词(q)、代词(r)、时间词(t)
- 虚词：副词(d)、介词(p)、连词(c)、助词(u)、标点(w)

## 🛠️ 技术栈

- **后端**：Python 3.7+, Flask 3.0.0
- **数据库**：SQLite + SQLAlchemy
- **NLP处理**：jieba（分词）, SnowNLP（情感分析）
- **前端**：Bootstrap 5, Chart.js
- **架构**：MVC 模式，RESTful API

## 🚀 快速开始

### 环境要求
- Python 3.7 或更高版本
- pip（Python包管理工具）
- 现代浏览器（推荐 Chrome、Firefox、Edge）

### Windows 安装步骤

```bash

# 将项目文件夹放到合适位置，如 D:\text_annotation_system

# 2. 打开命令行，进入项目目录
cd D:\text_annotation_system
