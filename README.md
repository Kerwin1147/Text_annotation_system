# 文本标注系统

一个基于Flask的智能文本标注系统，支持中文分词、词性标注和命名实体识别（5种实体类型）。

## 功能特性

### 核心功能
- **文本管理**：支持文件上传和手动输入，任务列表管理
- **智能标注**：自动分词、词性标注、命名实体识别、文本分类、情感分析
- **手动标注**：双模式标注（分词模式/实体模式），支持词性修改、实体标注、词语合并
- **知识库管理**：自动学习实体、手动添加、搜索、导出功能
- **数据统计**：可视化图表展示标注进度和实体分布

### 支持的命名实体类型
- 👤 人名：张三、李四、王五
- 📍 地名：北京、上海、长江
- 🏢 组织机构：清华大学、阿里巴巴
- 📅 时间日期：2024年、今天、明天
- 💲 数值金额：100元、1000万

## 技术栈

- Python 3.7+
- Flask 2.3.3
- SQLAlchemy 2.0.20
- SQLite
- jieba 0.42.1（中文分词）
- SnowNLP 0.12.3（情感分析）
- Bootstrap 5.3.0
- Chart.js

## 系统要求

- Python 3.7 或更高版本
- pip（Python包管理工具）
- 现代浏览器（推荐Chrome、Firefox、Edge）
